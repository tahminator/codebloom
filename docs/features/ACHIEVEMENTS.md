Announcements are automatically provided to a given user at the end of a leaderboard. A user can receive an announcement for the following condition(s):

- If a user is in the top 3 of the global leaderboard against all users
- If a user is in the top 3 of any university/club/organization leaderboard against it's users
  - A user can win multiple awards in this category from one leaderboard (one per filter).
